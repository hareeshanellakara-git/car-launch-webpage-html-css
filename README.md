# Lamborghini Aventador SVJ – Launch Website

This project is a static car launch webpage designed using pure HTML and CSS.

Features:
- Rotating company logo using CSS animation
- Dynamic background color transitions
- Hover-based image scaling
- Styled feature descriptions
- Moving “Book Now” call-to-action button
- Neon-style typography

Technologies used:
- HTML5
- CSS3 (Animations, Transforms, Transitions)

Purpose:
Academic project and frontend practice.
